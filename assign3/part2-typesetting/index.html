Matthew’s Typeset Homepage

Welcome

This page demonstrates clean typesetting using semantic HTML and CSS. We combine a readable serif body with bold, condensed headings for a trustworthy look.

Typography choices, link behavior, lists, and navigation have all been tuned to match the assignment’s rubric.

Features

Below are a few highlights of the styles used on this page.

  • Two complementary web fonts (body + headings)
  • Comfortable line length and line height
  • Accessible link styles with visible focus
  • Lists spaced to align with paragraph rhythm
  • A flexbox navigation bar that forms a top/bottom rule
